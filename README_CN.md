一键下载Bing(必应)的每日壁纸，避免水印。

# 使用 #
	$ python Wallpaper_downloader.py

# 要求 #
最好使用python3，同时设置一个目录以存储壁纸，类似'/home/usrname/Pictures/BingWallpapers'
然后修改Wallpaper_downloader.py第14行的self.save_fold为同样值就可以了。
